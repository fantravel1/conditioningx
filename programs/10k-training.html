8-Week 10K Training

Double your distance. Go from 5K runner to confident 10K finisher.

8

Weeks

4x

Per Week

Prerequisite

Prerequisites

This program assumes you can comfortably run a 5K (3.1 miles). If you can't yet, complete the Couch to 5K program first.

Program Overview

This progressive program builds weekly mileage while introducing speed work. You'll run 4 days per week with 3 rest/cross-training days.

Week 1-2: Base Building

Weekly Mileage: 12-15 miles

  • Day 1: Easy run - 3 miles
  • Day 2: Rest or cross-training
  • Day 3: Easy run - 3 miles
  • Day 4: Rest
  • Day 5: Easy run - 2.5 miles
  • Day 6: Long run - 4 miles (easy pace)
  • Day 7: Rest

Week 3-4: Building Endurance

Weekly Mileage: 16-18 miles

  • Day 1: Easy run - 3.5 miles
  • Day 3: Tempo run - 3 miles (1 mile easy, 1 mile tempo, 1 mile easy)
  • Day 5: Easy run - 3 miles
  • Day 6: Long run - 5 miles

Week 5-6: Speed Introduction

Weekly Mileage: 18-22 miles

  • Day 1: Easy run - 4 miles
  • Day 3: Intervals - 6x400m at 5K pace with 90s recovery
  • Day 5: Easy run - 3.5 miles
  • Day 6: Long run - 6 miles

Week 7-8: Race Prep

Week 7: Peak week - longest long run (7 miles)

Week 8: Taper week - reduce mileage 30-40%

Race Day: Start easy, negative split the second half!
