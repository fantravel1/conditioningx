5K to 10K Program

Double your distance in 6 weeks. From 5K finisher to 10K runner.

6

Weeks

4x

Per Week

30-60

Min/Session

10K

Goal

Prerequisites

  • Can run 5K (3.1 miles) without stopping
  • Running at least 3x per week consistently
  • Comfortable running 30 minutes continuously
  • No current injuries

Program Structure

This program adds distance graduallyâ€”no more than 10% per week. You'll run 4 days per week, with one long run that progressively builds to 10K distance.

Week 1

  • Day 1: Easy run - 25 min
  • Day 2: Rest or cross-train
  • Day 3: Tempo run - 20 min (comfortably hard)
  • Day 4: Rest
  • Day 5: Easy run - 25 min
  • Day 6: Long run - 35 min
  • Day 7: Rest

Weekly total: ~105 minutes running

Week 2

  • Day 1: Easy run - 30 min
  • Day 2: Rest or cross-train
  • Day 3: Intervals - 5 min easy, 4x(3 min fast/2 min easy), 5 min easy
  • Day 4: Rest
  • Day 5: Easy run - 25 min
  • Day 6: Long run - 40 min
  • Day 7: Rest

Weekly total: ~115 minutes running

Week 3

  • Day 1: Easy run - 30 min
  • Day 2: Rest or cross-train
  • Day 3: Tempo run - 25 min
  • Day 4: Rest
  • Day 5: Easy run - 30 min
  • Day 6: Long run - 45 min
  • Day 7: Rest

Weekly total: ~130 minutes running

Week 4

  • Day 1: Easy run - 30 min
  • Day 2: Rest or cross-train
  • Day 3: Intervals - 5x(4 min fast/2 min easy)
  • Day 4: Rest
  • Day 5: Easy run - 30 min
  • Day 6: Long run - 50 min
  • Day 7: Rest

Weekly total: ~140 minutes running

Week 5

  • Day 1: Easy run - 35 min
  • Day 2: Rest or cross-train
  • Day 3: Tempo run - 30 min
  • Day 4: Rest
  • Day 5: Easy run - 30 min
  • Day 6: Long run - 55-60 min (approaching 10K distance)
  • Day 7: Rest

Weekly total: ~155 minutes running

Week 6 - Race Week

  • Day 1: Easy run - 25 min
  • Day 2: Rest
  • Day 3: Easy run - 20 min with 4x30 sec pickups
  • Day 4: Rest
  • Day 5: Rest or very easy 15 min jog

  • Day 7: Rest and celebrate

Taper week - reduced volume, fresh legs for race day

Training Tips

Easy Runs

Should feel truly easy. You should be able to hold a conversation. If you can't talk, slow down. These build aerobic base.

Tempo Runs

"Comfortably hard" - you can speak in short phrases but not sentences. About 80-85% effort.

Intervals

Fast portions at 85-90% effort. Recovery portions are truly easy. These build speed and VO2 max.

Long Runs

Slow and steady. The goal is time on feet, not pace. Slower than your easy runs is fine.
